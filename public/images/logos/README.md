# Company Logos

Place all company logo images in this directory. The logos should be:

- PNG format with transparent background (recommended)
- Minimum width: 200px
- Aspect ratio: Maintain original logo proportions
- File naming: Use lowercase with hyphens (e.g., `air-india.png`, `qatar-airways.png`)
